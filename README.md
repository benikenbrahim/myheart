**MyHeart** est un système de gestion des soins de santé moderne basé sur une **architecture microservices**. Il permet de gérer efficacement les patients, les rendez-vous, la facturation, les analyses de laboratoire et les notifications.


Technologies utilisées:

- **Backend :** Spring Boot (Java), Node.js/Express, Python/FastAPI
- **Bases de données :** PostgreSQL, MongoDB
- **Communication :** API Gateway (ESB)
- **Conteneurisation :** Docker
- **Tests :** Postman
